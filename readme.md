### Prerequisites
Make sure you have the following installed on your machine:
- **Node.js** (LTS version recommended)
- **Angular CLI**
- **MongoDB** (if using a local database)

### Backend Setup
```sh
cd backend
npm install
npm start
```

### Frontend Setup
```sh
cd frontend
npm install
ng serve
```

Your application should now be running at `http://localhost:4200/`
